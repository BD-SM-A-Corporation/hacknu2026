## Описание системы
RailTelemetry предоставляет мощные инструменты для контроля и аналитики локомотивов:
* Мониторинг в реальном времени с использованием WebSocket.
* Архив телеметрии с агрегацией данных.
* Умные графики с возможностью масштабирования.
